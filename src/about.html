    • Timer

    Timer is a simple website created using p5.js. This website lets you set a timer and once it is done, it will make a sound to let you know that your time is up!

    This website is a PWA, which means you can install this website as app and use it offline whenever you want. Just save/install this website to your device. You can find basic instructions to install this website here.

    This project is complete open source project. You can find the code for this project on GitHub. Any contributions are most welcomed.

    Thank you for visiting our website.
